
## Supported Extensions
- I: Fully
- M: Fully
- A: Fully
- F: Fully
- D: Fully

- Zb*: part
- Zfh: not

## Abbreviation

- imm:    immediate
- uimm:   unsigned immediate
- zimm:   zero-extended immediate

- shamt:  shift amount

- off:    offset
- rd=x0:  If rd is omitted, x0 is assumed.
